# Frequently Asked Questions

Below are answers to questions you might have.

## What is `Ultimate Block Breaker`?

**Ultimate Block Breaker** is a Minecraft mod, datapack and Add-On that adds tiered, upgradeable block breaker blocks that automatically break most blocks for you (great for things like automating cobblestone generators). It includes multiple material tiers (wood to netherite) withy varying speeds and upgrade items to enhance efficiency.
