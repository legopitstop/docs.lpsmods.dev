# How to Find Pack Info

Step-by-step instructions on how to find the pack information for [Minecraft: Bedrock Edition](https://minecraft.wiki/w/Bedrock_Edition)

![Pack Info](https://cdn.lpsmods.dev/data/docs/015a68ef-5b4d-4c92-a9be-f3a96e5511ec.webp)

The Pack Info menu can provide the following information:

| Name            | Description                                                                                                      | Example                                |
| --------------- | ---------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| `PACK ID`       | The pack's unique ID. All packs have a different ID. If a pack has the same ID, it will cause overriding errors. | `cb00feac-550e-4d01-a7bf-c6b4e23d622b` |
| `PACK VERSION`  | The installed version of the pack.                                                                               | `1.6.3`                                |
| `FILE LOCATION` | The location of the pack on your device.                                                                         |                                        |
| `ERROR LIST`    | A list of any errors the pack is having. If you see errors contact the dev of the pack.                          |                                        |

:::tabs

== OreUI

1. Open Minecraft Bedrock Edition
2. Press "Play" to open worlds menu.
   ![step2](https://cdn.lpsmods.dev/data/docs/14a38336-e7ee-4ae0-adb4-759c14240c65.webp)

3. Edit the world that you want to see the add-on's info
   ![step3](https://cdn.lpsmods.dev/data/docs/f6a325bf-0f25-47b8-80aa-86022ca70493.webp)

4. Press "Behavior packs" or "Resource packs"
   ![step4](https://cdn.lpsmods.dev/data/docs/1e25bd59-aac0-4650-9282-eb25b1e44ee3.webp)

5. Expand the pack description.
   ![step5](https://cdn.lpsmods.dev/data/docs/4e3e762b-52cb-4776-b575-c831b64323e8.webp)

6. Finally, press "Technical details" to open pack details.
   ![step6](https://cdn.lpsmods.dev/data/docs/fdef21d7-c0a0-42e4-ab50-25a253a0f23e.webp)

== Legacy UI

1. Open Minecraft Bedrock Edition
2. Press "play" to open worlds menu.
   ![step2](https://cdn.lpsmods.dev/data/docs/1276f0ba-7e72-4aac-8724-2fe95eece56c.webp)

3. Edit the world that you want to see the add-on's info
   ![step3](https://cdn.lpsmods.dev/data/docs/908b15e5-6709-42a4-9143-378e6a3424ec.webp)

4. Press "Behavior Packs" or "Resource Packs"
   ![step4](https://cdn.lpsmods.dev/data/docs/b7073fd7-7eb1-4628-b2fe-c01027b70248.webp)

5. Press "ACTIVE PACKS" to show all the activated packs.
   ![step5](https://cdn.lpsmods.dev/data/docs/3e5c55f8-6774-45ad-9b04-54da6eab8948.webp)

6. Now select the pack you want to check.
   ![step6](https://cdn.lpsmods.dev/data/docs/641b7d83-4783-4083-987b-8c6cfe7a5944.webp)

7. Finally, click the green check to open the Pack Validation menu.
   ![step7](https://cdn.lpsmods.dev/data/docs/7561b7e2-504f-407c-9a41-549bf753705c.webp)

:::
