# How to Find com.mojang Folder

<!-- TODO: Needs screenshots -->

Step-by-step instructions on how to find the `com.mojang` folder for [Minecraft: Bedrock Edition](https://minecraft.wiki/w/Bedrock_Edition)

:::tabs key:platform

== Windows

:::tabs

=== GDK

1. Press `Win + E` to open File Explorer.
2. Type the below path into the address bar and press Enter.

   ```txt
   %appdata%\Minecraft Bedrock\Users\Shared\games\com.mojang
   ```

3. This will take you directly to the com.mojang folder.

=== UWP (legacy)

1. Press `Win + E` to open File Explorer.
2. Type the below path into the address bar and press Enter.

   ```txt
   %localappdata%\Packages\Microsoftcom.mojangUWP_8wekyb3d8bbwe\LocalState\games\com.mojang
   ```

3. This will take you directly to the com.mojang folder.

== Linux

:::tabs

=== minecraft-linux

1. Open your file manager.
2. Press `Ctrl + H` to show hidden files.
3. Navigate to `~/.var/app/io.mrarm.mcpelauncher/data/mcpelauncher/games/com.mojang`.
4. This will take you directly to the com.mojang folder.

== iOS

1. Open Files
2. In the Browse tab, tap `On My iPhone`
3. Press `Minecraft`
4. Then tap `games`
5. Finally, tap `com.mojang`

== Android

:::tabs

=== External

1. Install and open [Google Files](https://play.google.com/store/apps/details?id=com.google.android.apps.nbu.files)
2. Tap `Internal Storage`
3. Tap `data`
4. Tap `com.mojang.minecraftpe`
5. Tap `files`
6. Tap `games`
7. Finally, tap `com.mojang`

=== Application

You cannot access your com.mojang folder in Application storage location.

:::
