# Sheath

<InvSlot id="barked:bamboo_sheath"/>

**Sheath** is an item you get when you strip a bamboo block.

## Obtaining

Using an axe on a bamboo block causes it to become a stripped bamboo block, dropping 2-4 sheaths or 4-6 for wood.
