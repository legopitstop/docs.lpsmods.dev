# Frequently Asked Questions

Below are answers to questions you might have.

## What is `Lightning Bolt Glass`?

**Lightning Bolt Glass** is a Minecraft mod, datapack and Add-On that adds a mechanic where lightning transforms sand into glass (called fulgurites) when it strikes, making storms more impactful and creative. It supports data-driven fulgurites so you can customize what blocks are created when lightning hits.
