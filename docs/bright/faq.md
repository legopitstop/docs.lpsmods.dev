# Frequently Asked Questions

Below are answers to questions you might have.

## What is `Bright`?

**Bright** is a Minecraft mod, datapack and Add-On that adds a set of unique blocks to the game to brighten up your world. It includes five types of lights with all 16 color variants plus metal variations, giving you decorative and functional illumination options beyond vanilla lighting.
