# Frequently Asked Questions

Below are answers to questions you might have.

## What is `More NBT`?

**More NBT** is a Minecraft mod, datapack and Add-On that adds additional custom NBT (Named Binary Tag) functionality to items and objects, giving creators more control for events and behaviors via commands. It serves largely as an API for custom tags that lets you trigger events (like on use/hold), custom durability, animated models, throwable items, and more advanced item behavior for other datapacks and customization projects.
