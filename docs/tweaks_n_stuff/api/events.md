# Added Events

Tweaks & Stuff adds a few Blockbench events.

| Event ID          | Description                     |
| ----------------- | ------------------------------- |
| `profile_changed` | User has changed their profile. |
| `show_dialog`     | A dialog has been opened.       |
| `hide_dialog`     | A dialog has been hidden.       |
