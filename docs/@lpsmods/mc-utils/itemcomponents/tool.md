# mcutils:tool

Deals damage to the item when you break a block or hit an entity.

## Component

### Options

| Name                | Type    | Default | Description |
| ------------------- | ------- | ------- | ----------- |
| `damage_when_mined` | boolean | `false` |             |
| `damage_when_hit`   | boolean | `false` |             |
