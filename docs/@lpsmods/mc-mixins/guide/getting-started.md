# Getting Started

## Using it in your project

To use `@lpsmods/mc-mixins` in your project you'll need to bundle and merge it with your packs code. You can do this with [esbuild](https://esbuild.github.io/getting-started/#your-first-bundle)

A bundled build of mc-mixins can be downloaded from the Releases page on GitHub.
